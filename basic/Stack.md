# 스택

> LIFO(Last In, First Out)을 만족하는 자료구조

알고리즘에 갓 입문한 사람들은 스택을 직접 만들어서 풀려고 하는 경우가 많은데, STL에서 제공하는 스택을 쓰는 것이 좋다. 훨씬 안정적이고 편리하다.

아래 문제에서 스택을 어떻게 쓰는지 알 수 있다.
